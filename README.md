# Hermes Subagent Ops

Hermes 서브에이전트, 분신술, Prime 중심 운영 모델을 정리한 공용 레포입니다.

이 레포는 다음을 목표로 합니다.
- Hermes가 root agent(Prime)로 동작하는 운영 원칙 정리
- 분신(worktree clone) 기반 병렬 실행 체계 문서화
- 24시간 일하는 자율 에이전트 운영 패턴 정리
- 완료 판단, 검증 게이트, 보고 포맷 표준화

핵심 원칙
- Prime만 완료를 선언한다
- 분신은 실행만 하고 결과/증거만 올린다
- 말뿐인 분신이 아니라 실제 worktree/branch/state를 가진 물리 분신을 쓴다
- 24/7 자율실행은 단순 반복이 아니라 변경 → 결과 측정 → 학습 루프로 운영한다

구성
- `SKILL.md`: 공용 분신술 운영 스킬 본문
- `docs/prime-gate.md`: Prime 승인 게이트와 완료 상태 머신
- `docs/24x7-loop.md`: 24시간 자율실행 운영 루프

설명
이 레포는 "Hermes 서브에이전트에 대한 운영 스킬"을 담은 저장소이며,
실제로 24시간 일하는 에이전트 시스템을 어떻게 설계하고 굴릴지에 초점을 둡니다.
